# GPA Calculator (حاسبة المعدل التراكمي)

A modern, responsive, and legendary GPA calculator application built with React, Vite, and Tailwind CSS.
Features full Arabic localization, RTL support, and PDF export functionality.

## Features
- **Semester GPA**: Calculate GPA for the current semester.
- **Cumulative GPA**: Calculate updated cumulative GPA.
- **PDF Export**: Download detailed grade reports with Arabic support.
- **Dark Mode**: Sleek, high-contrast UI.

## Local Development
1. Clone the repo
2. `npm install`
3. `npm run dev`
